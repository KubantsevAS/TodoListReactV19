## Getting Started

1. Clone the repo

    ```git
    git clone https://github.com/KubantsevAS/TodoListReactV19.git
    ```

2. Install all dependencies

    ```sh
    yarn
    ```

3. Run json-server

    ```sh
    yarn run server
    ```

4. Run the development server

    ```sh
    yarn run dev
    ```
